第一步

安装node环境 可查看node

以linux系统为例：

我这里使用的 nvm node版本管理器

  • 首先下载nvm
    wget -qO- https://raw.githubusercontent.com/creationix/nvm/v0.33.2/install.sh | bash

安装完成之后

  • 配置nvm环境变量

    vim ~/.bash_profiles

  • 粘贴线代码到文件末尾

    export NVM_DIR="$HOME/.nvm"
[ -s "$NVM_DIR/nvm.sh" ] && . "$NVM_DIR/nvm.sh" # This loads nvm

  • 执行 生效环境变量

    source ~/.bash_profiles

验证 nvm 是否安装成功

  • 执行nvm
    $ nvm --verion
0.33.0
    打印版本号，说明安装成功
